Atlassian is the most profitable technology company in the history of Australia. Among other things, they produce a variety of tools for team collaboration and project management. Some of their tools are:



  1. Jira: a project and issue tracking tool for software and business teams
  2. Jira Service Management (JSM): a helpdesk tool that integrates the best of ITSM principles
  3. Confluence: a collaborative document and knowledge management tool
  4. Trello: a visual project management tool that uses a Kanban-style board
  5. Bamboo: a continuous integration and deployment product that offers automated integration
  6. Bitbucket: a distributed version control system that supports Mercurial and Git
  7. Crucible: a collaborative code review application


Atlassian Standard Infrastructure (ASI) is a set of best practices and guidelines for running Atlassians software products in a secure and scalable way. It includes recommendations for:

  1. Operating system and platform versions
  2. Database and application server configurations
  3. Network and security settings
  4. Performance and scalability configurations
  5. Backup and disaster recovery configurations

It's important to note that this is not a product, but a set of guidelines and recommendations for deploying and maintaining Atlassian software.
